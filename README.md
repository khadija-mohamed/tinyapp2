## Dependencies

- Node.js
- Express
- EJS
- bcrypt
- body-parser
- cookie-session


## Getting started | Project Documentation
- Firstly, install all dependencies listed using the `npm install` command.
- Lastly, run the server using the `node express_server.js` or `npm start` commands.

**Enjoy!**
